# libFilum Requirements

## Functional Requirements
1. read file.  
>1.1. Read entire file.  
>1.2. Read by line.  
>1.3. Read by data structure.  
2. write file.  
>2.1. write entire file.  
>2.2. write line.  
>2.2.1. overwrite line.  
>2.2.2. insert line.  
>2.3. Write data structure element.  

3. Cross platform
4. simple to add more formats.
